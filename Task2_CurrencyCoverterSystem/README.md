# Currency Converter (Java + Swing)

This is a desktop-based Currency Converter application built using Java. It allows users to convert currencies in real-time using exchange rates fetched from an API.

## Features
- Select base currency and target currency  
- Enter amount to convert  
- Real-time exchange rate fetching using API  
- Instant conversion result  
- Simple and user-friendly GUI using Java Swing  

## Learning Outcomes
- API integration in Java  
- GUI development using Swing  
- Handling user input and validation  
- Building a complete desktop application
