# ATM System 

This is a desktop-based ATM System application built using Java. It simulates basic ATM operations like withdrawing, depositing, and checking account balance through a graphical user interface.

## Features
- Withdraw and deposit money  
- Check account balance  
- Input validation for transactions  
- Multi-screen navigation using CardLayout  
- Simple GUI using Java Swing  

## Workflow
- The application starts with a Main Menu screen  
- User selects Withdraw or Deposit → navigates to Transaction screen  
- User enters amount → system processes transaction and shows result  
- Back button returns to Main Menu with cleared data  
- Check Balance shows current balance in a dialog  
- Exit closes the application
