# CODSOFT Java Development Internship

This repository contains all the tasks completed as part of my Java Development Internship at CODSOFT. Each project focuses on applying core Java concepts to build real-world applications.

# Task 1: Student Management System
A desktop application built using Java and Swing to manage student records.

Features:
- Add, update, delete, and search students  
- Display all student records  
- GUI-based interaction  
- Basic data handling and validation  


# Task 2: Currency Converter
A Java application that converts currencies using real-time exchange rates from an API.

Features:
- Select base and target currencies  
- Enter amount to convert  
- Fetch real-time exchange rates  
- GUI using Java Swing  


# Task 3: ATM System
A desktop-based ATM simulation application with multi-screen navigation.

Features:
- Withdraw and deposit money  
- Check account balance  
- Input validation for transactions  
- Navigation between screens (Menu ↔ Transaction)  
- Clean and simple GUI 

Learning Outcomes
- Object-Oriented Programming in Java  
- GUI development using Swing  
- API integration and real-time data handling  
- Structuring multi-class applications  
- Handling user input and validation
